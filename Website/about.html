Tentang Perusahaan & Studi Kasus

PT Vestia Cinta Abadi (VCA) adalah perusahaan digital entertainment dengan kebutuhan jaringan kompleks yang mencakup manajemen, produksi konten, server publik, serta layanan internal berbasis DNS, HTTP, SMTP, dan NAT.

Profil Perusahaan

PT Vestia Cinta Abadi (VCA) bergerak di industri virtual content creation dan menaungi sejumlah virtual streamer populer di Indonesia. Pertumbuhan talenta dan tim produksi membuat VCA melakukan ekspansi infrastruktur dengan membangun dua gedung utama:

  • Gedung 1 — Manajemen & Administrasi: PC Office, PC Admin, server Finance/HR, serta WAP untuk tamu.
  • Gedung 2 — Studio & Produksi: Data Center, PC Design, PC Ops, PC Stream, serta server Desain dan server Streaming.

Studi Kasus Jaringan

Studi kasus ini diambil langsung dari spesifikasi Tugas Besar II2120 Jaringan Komputer ITB. Sistem jaringan PT VCA mencakup empat subnet internal (A, B, C, D) dan satu subnet DMZ untuk server publik.

Subnet A & B (Gedung 1)

Office, Admin, Finance server, dan WAP tamu. Bersifat private dan tidak dapat diakses publik.

Subnet C & D (Gedung 2)

PC Ops, PC Design, PC Stream, serta server desain dan server streaming. Private & isolated.

DMZ Server Publik

3 server publik di Data Center yang diakses melalui NAT statis menggunakan blok publik 182.24.67.96/29.

Layanan yang Dikembangkan

  • NAT: Static NAT untuk server DMZ, PAT untuk subnet A–D.
  • DNS: Hierarki multi-subnet dengan domain internal dan delegasi zona.
  • SMTP: Email corporate (vca.co.id) untuk komunikasi antar-karyawan.
  • HTTP: Web server internal dan publik.
  • ACL: Pembatasan akses antar-subnet dan dari internet.
